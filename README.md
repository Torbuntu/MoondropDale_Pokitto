## Crop types and growth data

type-growth

0-0 : empty, no crop no water no till
0-1 : empty, but tilled for crop

T-2 : Seed is planted, any type
T-3 : Sappling is growing 

1-4_9 : turnip 
2-4_9 : radish
3-4_9 : daisy
4-4_9 : Green bean
5-4_9 : Coffee
6-4_9 : Tea
7-4_9 : Tomato
8-4_9 : Blueberry
9-4_9 : Magic Fruit
